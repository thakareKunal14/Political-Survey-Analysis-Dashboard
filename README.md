# Political-Survey-Analysis-Dashboard
An interactive Excel dashboard project built using Pivot Tables, Charts, Slicers, and VBA to analyze political survey data across Maharashtra. The dashboard provides insights into voter behavior, regional support trends, political party performance, key election issues, and urban vs rural analytics through dynamic visualizations and KPI tracking. 
Skills Used:

Excel • Pivot Tables • Charts • VBA • Dashboard Design • Data Analysis • Data Visualization

# Project Summary

## 📌 Project Overview

## Political Survey Analysis Dashboard

Designed and developed an interactive Excel-based political survey analytics dashboard to analyze voter sentiment and election trends across Maharashtra. The project includes dynamic KPI cards, slicers, pivot charts, and VBA automation to provide real-time filtering and visual storytelling.

### Key Highlights:

- Interactive Excel Dashboard
- Automated Reporting using VBA
- Dynamic Charts & Pivot Tables
- Advanced Data Visualization
- Survey Trend Analysis
- Political & Demographic Insights

## 🎯 Key Objectives

- Analyze voter sentiment and political preferences.
- Identify leading political parties and voter trends.
- Compare urban vs rural voting behavior.
- Track survey trends over time.
- Highlight key election issues influencing voters.
- Create an interactive and professional dashboard for decision-making.

## 📊 Dataset Description

The dataset contains survey responses from approximately **2000 respondents** across different regions of Maharashtra.

### Key Fields Included:

- Respondent ID
- Region
- Gender
- Age Group
- Political Party Preference
- Government Rating
- Voting Reason
- Urban/Rural Classification
- Survey Month
- Swing Voter Indicator

## 🛠 Tools & Technologies Used

- **Microsoft Excel**
- **Pivot Tables & Pivot Charts**
- **Excel VBA (Macros & Buttons)**
- **Data Cleaning & Data Validation**
- **Interactive Slicers**
- **Dashboard Design & Data Visualization**

### 🔎 Key Finding:

- BJP emerged as the leading party among respondents.
- Inflation was identified as the top voter concern.
- Urban voters showed greater preference variation compared to rural voters.
- North Maharashtra recorded the highest support concentration.
- Swing voters played a major role in influencing election outcomes.

### 📌 Challenges Solved:

- Organized large survey datasets into structured analysis.
- Automated dashboard filtering using slicers and VBA.
- Designed a professional and visually appealing UI in Excel.
- Converted raw survey data into actionable insights.
